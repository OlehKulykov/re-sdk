52 /* README */
53 /*
54  The code contained here is Copyright (C) 2008-2009 Robin Watts
55  <robin@wss.co.uk>
56 
57  This code provides optimised ARM code (and the C equivalents) for fast
58  conversion of YUV420, YUV422 and YUV444 code to RGB565.
59 
60  For ARM platforms, use the ARM code, as this will result in significantly
61  faster results (due to no register spillage in the loops, improved
62  scheduling etc).
63 
64  For other platforms, use the C code - this is deliberately kept as close
65  as possible to the ARM code.
66 
67  yuv2rgb.h is the overall header. #include this.
68 
69  yuv2rgb16tab.c contains the definition of the standard table used by
70  all the routines (and passed in as a parameter). You can use this, or
71  define your own table.
72 
73  Future versions of this software may include routines to generate tables
74  with given weights.
75 
76  The latest version of this software should always be available from
77  <http://www.wss.co.uk/pinknoise/yuv2rgb>
78  */
