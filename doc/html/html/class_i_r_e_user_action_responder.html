Detailed Description

Class interface used for object that can receive user messages, clicks/touches. User action moves down from top object to parent in object hierarchy. For receiving events object and all parent objects must responds for user action IsRespondsForUserAction().

virtual REBOOL IREUserActionResponder::IsInterceptsUserAction ( ) const
pure virtual

Return value that indicates that object intercepts/block user action from parent objects. Default value is 'true', in this case scaning objects will be breaked on this object. If this value is 'false' than user action will be directed through this object and directed to parent objects.
