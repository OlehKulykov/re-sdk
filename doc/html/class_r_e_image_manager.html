Creates and return new buffer object with JPEG file presentation from image data(pixels), width, height and pixel format. This functionality requires undefined flag RE_RECORE_NO_JPEG_IMAGE_SUPPORT Use quality in range [0.0f; 100.0f]. Currently supported only RGB images(REImagePixelFormatR8G8B8 format).
